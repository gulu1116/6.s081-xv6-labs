
## MIT 6.S081 XV6 Labs


个人学习记录，仅供参考。每个实验的分支对应独立实验代码，可切换分支查看。所有实验均通过测试。详细思路见labsnote。


| 实验    | 名称                   | 简要说明                    |
| ----- | -------------------- | ----------------------- |
| Lab0  | Setup                | 项目环境搭建与基础配置             |
| Lab1  | XV6 & Unix Utilities | 编译运行 XV6 内核及基本工具程序      |
| Lab2  | System Calls         | 增加系统调用接口，实现用户程序调用内核功能   |
| Lab3  | Page Tables          | 学习内存页表及地址映射，修改页表管理      |
| Lab4  | Traps                | 实现异常处理与中断机制             |
| Lab5  | Lazy Page Allocation | 实现延迟分配内存页，优化内存使用        |
| Lab6  | Copy-on-Write Fork   | 实现写时复制的 fork 操作         |
| Lab7  | Multithreading       | 内核多线程实现与调度              |
| Lab8  | Locks                | 实现锁机制，保证并发安全            |
| Lab9  | File System          | 文件系统实验，理解 inode、目录与文件操作 |
| Lab10 | Mmap                 | 实现内存映射机制                |
| Lab11 | Network              | 基本网络实验，实现简单通信           |
